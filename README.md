# MS-GADGETS - Business Management Suite

A comprehensive single-page application for managing gadget/electronics sales, inventory, and customer relationships.

## Features

- **Point of Sale (POS)** - Easy product checkout and transaction management
- **Inventory Management** - Track stock levels and calculate profits
- **Debtor Management** - Manage customer credit accounts and payment tracking
- **Sales Analytics** - View sales metrics, profits, and business performance
- **Communications Hub** - Chat system for team coordination
- **User Management** - Role-based access control (Admin/User)
- **Data Backup & Recovery** - Export/import system data

## Deployment

### Render Deployment

1. Connect your GitHub repository to Render
2. Create a new Web Service
3. Set the Build Command: `npm install`
4. Set the Start Command: `npm start`
5. Deploy!

### Local Development

```bash
npm install
npm start
```

Access the app at `http://localhost:3000`

## Technology Stack

- **Frontend**: HTML5, Tailwind CSS, Lucide Icons
- **Backend**: Node.js, Express
- **Storage**: LocalStorage (client-side)
- **Icons**: Lucide
